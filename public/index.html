Microsoft Graph API Reference

Ready-to-use queries for Settings Catalog data from your tenant

List all Settings Catalog policies

GET https://graph.microsoft.com/beta/deviceManagement/configurationPolicies

Search Settings Catalog definitions by name

GET https://graph.microsoft.com/beta/deviceManagement/configurationSettings
  ?$filter=contains(name,'{KEYWORD}')
  &$select=id,name,description,settingDefinitionId,applicability,defaultValue,categoryId,options
  &$top=50

Create Settings Catalog policy (POST)

POST https://graph.microsoft.com/beta/deviceManagement/configurationPolicies
Content-Type: application/json

{
  "name": "My Policy",
  "platforms": "windows10",
  "technologies": "mdm",
  "settings": [{
    "@odata.type": "#microsoft.graph.deviceManagementConfigurationSetting",
    "settingInstance": {
      "@odata.type": "#microsoft.graph.deviceManagementConfigurationChoiceSettingInstance",
      "settingDefinitionId": "device_vendor_msft_...",
      "choiceSettingValue": { "value": "..._1", "children": [] }
    }
  }]
}
